# Multi-Tenant CRM Frontend

A modern SaaS CRM frontend built using Next.js, TypeScript, and Tailwind CSS.  
This project is part of a full-stack multi-tenant CRM system designed for managing leads, customers, tasks, and sales workflows.

---

## 🚀 Features

- Multi-tenant SaaS architecture
- Authentication & Role-Based Access Control (RBAC)
- Lead management system
- Customer management
- Sales pipeline tracking
- Task & activity management
- Notes & follow-ups
- Dashboard analytics
- AI-assisted sales workflows
- Responsive modern UI

---

## 🛠️ Tech Stack

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS
- Zustand
- Axios
- shadcn/ui
